## Typography

### Headings

Use a premium editorial serif font similar to:

```css
font-family: "Cormorant Garamond", serif;
```

or

```css
font-family: "Playfair Display", serif;
```

Requirements:

* Uppercase text (TEXT-TRANSFORM: UPPERCASE)
* Bold appearance
* Strong letter spacing
* Large typography
* Premium editorial style

Example:

```css
text-transform: uppercase;
font-weight: 700;
letter-spacing: 0.08em;
```

---

### Body Text

```css
font-family: "Inter", sans-serif;
```

Requirements:

* Clean
* Modern
* Easy to read

## Layout Style

The website should feel:

* Premium
* Minimalist
* Editorial magazine
* Specialty coffee brand
* Japanese-inspired
* Elegant

Use:

* Large whitespace
* Large hero typography
* Clean grid layout
* Soft shadows
* Rounded corners
* High-quality imagery

---

## Important

The visual appearance should be very close to Blue In Green Coffee Roasters:

* White background dominant
* Navy blue typography
* Large uppercase headings
* Elegant serif fonts
* Premium editorial layout
* Spacious design with lots of whitespace

Keep all existing functionality unchanged.
Only redesign the visual layer (HTML/CSS/UI).
